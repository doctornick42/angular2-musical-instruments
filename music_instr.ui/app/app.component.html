Angular 2 musical instruments

Here we go. This app was made to discover some Web Audio API features. Also, this is my 'Hello World' in Angular 2 and Typescript worlds.

Right now there are 3 instruments here: a simple monophonic synthesizer, a polyphonic one, both with a regular piano keyboard. And "experimental" one, I call it a "plot synth".

It's easy to hear that there are some bugs here: for example, I'm pretty sure that attack/release parameters don't work correctly. Anyway, later I will fix it when I add new parameters and concentrate on a sound process.
